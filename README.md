## 📖 前 言
我的原插件：云崽机器人-[R插件](https://gitee.com/kyrzy0416/rconsole-plugin)的大部分核心功能都会转移到这里
> 我深知云崽插件的弊端，无法更好的支持我的插件，决定不再更新，转向koishi，其中一个原因是TypeScript > JavaScript，因为它有更好的类型检测
